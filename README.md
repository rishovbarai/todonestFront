# Todo Frontend

Nuxt.js frontend for the Todo application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:3000`

## Configuration

- API Base URL: Configured in `nuxt.config.ts` (default: `http://localhost:3001`)
- Can be overridden with `API_BASE_URL` environment variable

## Project Structure

- `pages/` - Application pages
- `composables/` - Reusable composables (API calls, etc.)
- `assets/scss/` - Styles and variables
- `components/` - Vue components (to be added)
